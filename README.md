# KitsX Custom

KitsX Custom is a lightweight Paper plugin that lets players create and edit custom PvP kits directly in‑game.  
Designed for KitPvP and practice servers, it provides a simple command system for fast kit editing without heavy menus or extra dependencies.

## ✨ Features
- Create and edit custom PvP kits in‑game  
- Simple command-based workflow  
- No menus, no dependencies  
- Fast and lightweight  
- Perfect for KitPvP and practice servers  

## 🧪 Commands
`/customkit` — Open the custom kit editor  
`/kitcancel` — Cancel your editing session  

## 🛠 Compatibility
- **Minecraft:** 1.21.x  
- **Platform:** Paper  

## 📦 Installation
1. Download the latest release from the **Releases** tab.  
2. Drop the `.jar` into your server’s `plugins` folder.  
3. Restart the server.  
4. Done — start editing kits.
